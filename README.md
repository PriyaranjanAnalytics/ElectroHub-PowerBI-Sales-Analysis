# ElectroHub Power BI Sales Analysis

## Project Overview
This Power BI project analyzes sales performance, profit trends, product performance, promotions, and regional sales insights for ElectroHub.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Key Features
- Top & Bottom 5 Product Analysis
- Sales Trend Analysis
- Profit vs Sales Insights
- City-wise Sales Map
- Promotion Analysis
- Period Comparison Dashboard
- Interactive Filters

## Data Transformation
- Replaced Null Values
- Created Conditional Columns
- Merged Queries
- Applied Left Join
- Created Calculated Columns

## DAX Calculations
Sales = Price Per Unit * Units Sold

Discount Value = Total Sales * Discount % / 100

## Dashboard Pages
1. Top Bottom 5 Analysis
2. Sales & Profit Insights
3. Period Comparison Dashboard

## Business Insights
- High-performing products identified
- Sales concentrated in major cities
- Promotions affected profitability
- Strong relationship observed between sales and profit
